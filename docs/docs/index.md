# Welcome to House Price MLOps

This documentation covers the engineering decisions, architecture, and code reference for the Housing Price Prediction system.

## Project Goals
1.  Build a robust regression model to predict housing prices.
2.  Automate the training and evaluation process.
3.  Deploy the model as a scalable Microservice on AWS.
